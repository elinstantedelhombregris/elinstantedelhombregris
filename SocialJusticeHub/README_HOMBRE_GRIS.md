*"Eres un pozo tallado no en piedra, sino en tiempo. Y estás destinado a desbordarte."*

## 🌟 Visión

Una plataforma digital donde cada argentino puede expresar sus sueños, valores y necesidades. Donde la transformación individual se convierte en transformación nacional. Donde el pensamiento sistémico, la amabilidad radical y la acción consciente se encuentran para crear la Argentina que merecemos.

## 🎯 Filosofía del Hombre Gris

El Hombre Gris no es una persona, es una idea que cobra vida. Es la encarnación de todo argentino que ha despertado del letargo colectivo y ha decidido que **¡BASTA!**

### Características Distintivas

- **Superinteligencia Sistémica**: Ve las relaciones ocultas entre fenómenos aparentemente desconectados
- **Amabilidad Radical**: Practica la amabilidad como ingeniería social
- **Liderazgo Distribuido**: No busca seguidores sino co-creadores
- **Diseño Idealizado**: Imagina el país como si pudiera crearse desde cero
- **Disolvedor de Problemas**: Crea herramientas que vuelven obsoletos los problemas
- **Transparencia Radical**: Su vulnerabilidad es su fortaleza

## 🗺️ Funcionalidades Principales

### 1. Mapa de Sueños, Valores y Necesidades
- **Sueños**: Comparte tus anhelos y esperanzas para un mejor futuro en Argentina
- **Valores**: Expresa los principios que crees que deberían guiar nuestro futuro
- **Necesidades**: Identifica qué necesitas tú o tu comunidad para prosperar

### 2. La Semilla de ¡BASTA!
- Árbol interactivo del movimiento
- Visualización de la estructura completa del cambio sistémico
- Cada nodo representa una acción, proyecto o logro ciudadano

### 3. Blog & Vlog
- Reflexiones profundas del Hombre Gris
- Análisis sistémico de los problemas argentinos
- Videos con conversaciones íntimas sobre transformación

## 🚀 Tecnologías

- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Mapas**: Leaflet.js para visualización geográfica
- **Estado**: React Query + Zustand
- **UI**: Radix UI + shadcn/ui
- **Backend**: Node.js + Express + PostgreSQL
- **Real-time**: WebSockets para actualizaciones en vivo

## 🎯 Objetivos del Movimiento

### La Tercera Oleada Inmigratoria
Convertir a Argentina en un país tan próspero, justo y amable que genere una tercera oleada inmigratoria. No de refugiados que huyen, sino de soñadores que vienen a construir.

### Interdependencia Consciente
Crear una nueva constitución que complemente la independencia política con la interdependencia consciente. Un pacto social donde cada derecho conlleva una responsabilidad.

### Amabilidad como Ley Invisible
Donde la cortesía cotidiana genera confianza y cooperación espontánea. Donde la amabilidad no es excepción sino norma.

## 🔧 Desarrollo

### Comandos Principales
```bash
# Instalar dependencias
npm install

# Desarrollo
npm run dev

# Build
npm run build

# Base de datos
npm run db:push

# Reseed cursos (incluye Masterclass Hombre Gris)
pnpm ts-node scripts/seed-courses.ts
```

> Cada vez que actualices contenido del curso “Teoría de Juegos Aplicada a Argentina: La Partida del Hombre Gris”, vuelve a ejecutar el comando de reseed para sincronizar las nuevas lecciones, gráficos y quizzes con la base local.
