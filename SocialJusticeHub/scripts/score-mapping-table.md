# Tabla de Asignación de Valores - Rueda de Vida

## Escala de Puntuación: 0-100 puntos

### Criterios de Evaluación por Tipo de Pregunta

#### Tipo 1: Preguntas de Frecuencia (¿Con qué frecuencia...?)
- **0-20 puntos**: Nunca / Casi nunca (0-1 vez al mes)
- **21-40 puntos**: Raramente (2-4 veces al mes)
- **41-60 puntos**: Ocasionalmente (1-2 veces por semana)
- **61-80 puntos**: Regularmente (3-5 veces por semana)
- **81-100 puntos**: Constantemente (Diario o casi diario)

#### Tipo 2: Preguntas de Calidad/Satisfacción (¿Qué tan satisfecho...?)
- **0-20 puntos**: Muy insatisfecho / Muy malo
- **21-40 puntos**: Insatisfecho / Malo
- **41-60 puntos**: Neutral / Regular
- **61-80 puntos**: Satisfecho / Bueno
- **81-100 puntos**: Muy satisfecho / Excelente

#### Tipo 3: Preguntas de Comportamiento Pasado (¿Cuándo fue la última vez...?)
- **0-20 puntos**: Hace más de 6 meses / Nunca
- **21-40 puntos**: Hace 3-6 meses
- **41-60 puntos**: Hace 1-3 meses
- **61-80 puntos**: Hace menos de 1 mes
- **81-100 puntos**: Esta semana / Muy reciente

#### Tipo 4: Preguntas de Obstáculos (¿Qué te impide...?)
- **0-20 puntos**: Muchos obstáculos / Muy difícil
- **21-40 puntos**: Varios obstáculos / Difícil
- **41-60 puntos**: Algunos obstáculos / Moderado
- **61-80 puntos**: Pocos obstáculos / Fácil
- **81-100 puntos**: Sin obstáculos / Muy fácil

#### Tipo 5: Preguntas de Impacto (¿Cómo afecta esto...?)
- **0-20 puntos**: Muy negativamente / Gran impacto negativo
- **21-40 puntos**: Negativamente / Impacto negativo moderado
- **41-60 puntos**: Neutral / Sin impacto significativo
- **61-80 puntos**: Positivamente / Impacto positivo moderado
- **81-100 puntos**: Muy positivamente / Gran impacto positivo

## Cálculo de Puntuación por Área

### Fórmula Base:
```
Puntuación Área = (Suma de puntuaciones de subcategorías) / (Número de subcategorías)
```

### Ponderación Opcional:
Si se quiere dar más peso a ciertas subcategorías:
```
Puntuación Área = Σ(Puntuación Subcategoría × Peso) / Σ(Pesos)
```

## Ejemplo de Cálculo

**Área: Salud**
- Ejercicio físico: 60 puntos
- Nutrición: 70 puntos
- Higiene: 90 puntos
- Salud mental: 50 puntos
- Sueño: 65 puntos

**Puntuación Total = (60 + 70 + 90 + 50 + 65) / 5 = 67 puntos**
