# Radar ¡BASTA! (PWA móvil) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** PWA móvil instalable (iOS/Android) en `/radar` para capturar las 6 señales de la Radiografía contra la misma base Neon.

**Architecture:** Ruta full-screen dentro del SPA existente (wouter + lazy), módulo de rutas Express nuevo (`routes-radar.ts`), manifest + service worker con scope `/radar`. Sin infra nueva.

**Tech Stack:** React 18 + wouter + framer-motion + design-tokens existentes; Express + Drizzle + Zod; PWA vanilla (manifest + SW a mano, sin vite-plugin-pwa).

**Spec:** `docs/superpowers/specs/2026-06-11-radar-pwa-design.md`

- [ ] **Step 2: Implementar el módulo.** Contenido clave:

```ts
export const radarSenalSchema = z.object({
  type: z.enum(['dream','value','need','basta','compromiso','recurso']),
  text: z.string().trim()
    .min(10, 'Contanos un poco más — al menos 10 caracteres')
    .max(1000, 'Máximo 1000 caracteres'),
  location: z.string().trim().max(255).optional(),
  latitude: z.coerce.number().min(-90).max(90).optional(),
  longitude: z.coerce.number().min(-180).max(180).optional(),
  category: z.enum([...categorias de userResources...]).optional(), // solo recurso
});
```

- `POST /api/radar/senal`: `radarSubmitRateLimit` (express-rate-limit, 12/15min por IP) + `optionalAuth` + `sanitizeInput`.
  - dream/value/need/basta → `storage.createDream({ [campoSegúnTipo]: text, type, location, latitude: String(lat), longitude: String(lng), userId: req.user?.userId ?? null })` (lat/lng son `text` en la tabla dreams).
  - compromiso sin sesión → 401 `{ message: 'Necesitás una cuenta para registrar compromisos', code: 'AUTH_REQUIRED' }`; con sesión → `storage.createUserCommitment(...)` (verificar firma en storage.ts; commitmentType `'initial'`, lat/lng `real`).
  - recurso sin sesión → 401 igual; con sesión → `storage.createUserResource({ description, category: category ?? 'other', ... })`.
  - Respuesta 201 `{ ok: true, type, id }`.
- `GET /api/radar/resumen`: cache en memoria 60s. Totales: `SELECT type, COUNT(*) FROM dreams GROUP BY type` (vía drizzle `db.select`), counts de `user_commitments` activos y `user_resources` activos; últimas 20 señales de `dreams` (id, type, extracto 140 chars, location, createdAt — sin userId). Respuesta `{ totals: {dream,value,need,basta,compromiso,recurso,total}, recientes: [...] }`.

- [ ] **Step 4: `main.tsx`** — reemplazar el unregister global:

```ts
if ('serviceWorker' in navigator) {
  const isRadar = window.location.pathname.startsWith('/radar');
  navigator.serviceWorker.getRegistrations().then(rs => {
    rs.forEach(r => {
      const scopePath = new URL(r.scope).pathname;
      if (!scopePath.startsWith('/radar')) r.unregister();
    });
  });
  if (isRadar) {
    window.addEventListener('load', () => {
      navigator.serviceWorker.register('/radar-sw.js', { scope: '/radar' }).catch(() => {});
    });
  }
}
```
