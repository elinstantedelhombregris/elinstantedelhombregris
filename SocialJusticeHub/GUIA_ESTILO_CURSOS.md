## Lineamientos de estilo para guías de estudio

- **Hilo conductor visible:** cada bloque debe citar la voz del Hombre Gris o la idea “jugamos el mismo juego” para mantener coherencia emocional.
- **Secciones escalonadas:** abrir con bloque hero (gradiente o color sólido) + cuerpo con títulos `<h3>` o `<h4>` claros + actividades/ejercicios.
- **Contraste obligatorio:** texto claro (`color: white` o `#f8fafc`) sobre fondos oscuros (`#0f172a`, `#111827`, `#1d4ed8`) y texto oscuro (`#0f172a`) para fondos claros.
- **Componentes reutilizables:** 
  - `div` hero con padding ≥ 2rem, bordes redondeados ≥ 12px y sombra sutil.
  - Grids responsivos (`display: grid; grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); gap ≥ 1.25rem`).
  - Tablas con encabezados resaltados (`background: #e0f2fe`) y separación clara.
  - Cajas de “Momento de revelación”, “Táctica psicológica”, “Actividad” con iconos/emoji contextuales.
- **Extensión objetivo:** mínimo 450–600 palabras por lección, combinando narrativa, ejemplos argentinos, tácticas psicológicas y ejercicios prácticos.
- **Lenguaje:** tono revelador, firme, esperanzador. Usar segunda persona y citas en cursiva para la voz del Hombre Gris.
- **Actividades guiadas:** cada lección debe cerrar con instrucción práctica (diario, role-play, canvas, simulación) que indique pasos numerados u ordenados.
- **Accesibilidad:** usar listas `<ul>`/`<ol>` con `line-height ≥ 1.8`, evitar bloques de texto mayores a 4 frases sin estructura.
- **Consistencia visual:** 
  - Iconografía textual (`🎯`, `🧠`, `📈`, etc.) homogénea.
  - Fondos oscuros ⇒ bordes transparentes; fondos claros ⇒ bordes `1px solid #e5e7eb`.
  - Siempre mencionar tiempo estimado (`duration`) coherente con densidad del contenido.
- **Mensaje final:** incluir citas directas del Hombre Gris en cajas destacadas y llamados a la reciprocidad Tit for Tat.

Usar estos lineamientos antes de actualizar o crear contenido para asegurar que las nuevas lecciones reflejen la estética y profundidad deseadas.
