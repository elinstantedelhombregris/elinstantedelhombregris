# CUADERNO DE SOMBRAS - ARCHIVOS PARA IMPRENTA

Este directorio contiene todos los archivos necesarios para la producción del "Cuaderno de Sombras" del movimiento El Instante del Hombre Gris.

## 📁 ESTRUCTURA DE ARCHIVOS

### Especificaciones Técnicas
- **00_ESPECIFICACIONES_TECNICAS.md**: Documento completo con todas las especificaciones técnicas para la imprenta

### Contenido por Sección
- **01_PORTADA_EXTERIOR.txt**: Contenido y diseño de la portada exterior
- **02_PORTADA_INTERIOR.txt**: Portada interior y página en blanco
- **03_PACTO_SOMBRA.txt**: Contrato de compromiso (Pacto de Sombra)
- **04_GLOSARIO.txt**: Glosario de términos del movimiento
- **05_INDICE.txt**: Índice completo del cuaderno
- **06_GUIA_USO.txt**: Guía de uso del cuaderno
- **07_RITUAL_SOMBRA.txt**: Plantilla de ritual de sombra
- **08_SIMBOLOS.txt**: Símbolos y marcas del cuaderno
- **09_MAPA_SEMANAS.txt**: Mapa visual de las 8 semanas
- **10_PRIMERA_ENTRADA.txt**: Primera entrada guiada
- **11_PROMPTS_SEMANALES.txt**: Prompts para las 8 semanas
- **12_PLANTILLA_REFLEXION.txt**: Plantilla de reflexión semanal
- **13_PLANTILLA_ESCRITURA_LIBRE.txt**: Plantilla de páginas de escritura libre
- **14_PAGINAS_FINALES.txt**: Páginas finales del cuaderno

### Referencias
- **15_REFERENCIA_COLORES.txt**: Referencia completa de colores en CMYK, RGB y Pantone
- **16_GUIA_DISENADOR.txt**: Guía completa para el diseñador gráfico

## 🎯 INFORMACIÓN GENERAL

### Formato
- **Tamaño**: A5 (148 × 210 mm)
- **Páginas**: 192 páginas (96 hojas)
- **Encuadernación**: Wire-o 19mm o rústica cosida
- **Papel**: Bond blanco 90 g/m² (interiores), Couché mate 250 g/m² (portada)

### Paleta de Colores
- **Gris Sabio**: #6B7280 (portada principal)
- **Celeste Argentino**: #74ACDF (acentos)
- **Escalas de gris**: 10%, 30%, 50%, 70% (interiores)

## 🚀 CÓMO USAR ESTOS ARCHIVOS

### Para Diseñadores Gráficos
1. Comenzar leyendo **00_ESPECIFICACIONES_TECNICAS.md**
2. Revisar **16_GUIA_DISENADOR.txt** para instrucciones detalladas
3. Usar los archivos de contenido (01-14) como referencia para cada página
4. Consultar **15_REFERENCIA_COLORES.txt** para colores exactos
5. Crear archivos PDF/X-4 con 300 DPI mínimo para imprenta

### Para Imprentas
1. Revisar **00_ESPECIFICACIONES_TECNICAS.md** para especificaciones técnicas
2. Verificar colores con **15_REFERENCIA_COLORES.txt**
3. Solicitar prueba de color física antes de impresión masiva
4. Seguir checklist de pre-impresión incluido en especificaciones

## 📋 CHECKLIST PRE-IMPRESIÓN

- [ ] Archivos en formato PDF/X-4
- [ ] Resolución mínima 300 DPI
- [ ] Sangrados de 3mm incluidos
- [ ] Tipografías embebidas o convertidas a curvas
- [ ] Colores en CMYK verificados
- [ ] Prueba de color física aprobada
- [ ] Numeración de páginas correcta (1-192)
- [ ] Márgenes y áreas seguras respetadas
