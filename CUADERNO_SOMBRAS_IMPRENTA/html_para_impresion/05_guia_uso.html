CÓMO USAR ESTE CUADERNO

  1. ESTABLECE TU RITUAL DE ESCRITURA
    Define un momento y lugar específicos para escribir. Puede ser al amanecer, antes de dormir, o cualquier momento que elijas. La consistencia es más importante que la duración.
  2. NO HAY REGLAS, SOLO COMPROMISO
    Este cuaderno es tuyo. Escribe lo que necesites, como lo necesites. No hay formato correcto o incorrecto. El único requisito es tu honestidad contigo mismo.
  3. USA LOS SÍMBOLOS PARA MARCAR ENTRADAS ESPECIALES
    Revisa la página de símbolos para aprender a marcar momentos de claridad, dudas, conexiones y revelaciones. Estos símbolos te ayudarán a navegar tu proceso.
  4. COMPARTE EXTRACTOS CUANDO TE SIENTAS CÓMODO
    El compartir es opcional pero poderoso. Puedes compartir con tu compañero de sombra, en círculos pequeños, o mantener todo privado. Tú decides tu nivel de apertura.
  1. PERMITE QUE EL PROCESO TE TRANSFORME
    No busques resultados específicos. Confía en el proceso. Las transformaciones más profundas ocurren cuando soltamos el control y permitimos que la sabiduría de las sombras nos guíe.
  2. HONRA TUS SOMBRAS
    Todo lo que escribas es válido. Las emociones difíciles, las dudas, las contradicciones son parte del proceso. No las juzgues, obsérvalas con curiosidad.
  3. MANTÉN EL COMPROMISO PERO SÉ GENTIL CONTIGO
    Si te saltas días, no te castigues. Simplemente retoma cuando puedas. El cuaderno está aquí para servirte, no para ser una carga.
  4. EXPERIMENTA CON FORMATOS
    Puedes escribir, dibujar, pegar objetos, hacer collages. Este cuaderno es un espacio creativo. Úsalo como necesites para expresar tu proceso.
