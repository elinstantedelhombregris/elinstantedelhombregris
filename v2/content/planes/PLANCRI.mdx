# PLANCRI — Crisis preparada

Argentina ha vivido crisis suficientes para haber aprendido a esperarlas. PLANCRI institucionaliza ese aprendizaje.

**Reservas estratégicas.** Alimentos, medicamentos esenciales, combustible, semillas — en cantidades calculadas por escenario, distribuidas geográficamente, rotadas para no degradarse.

**Sistemas de alerta temprana.** Salud (epidemiología activa), economía (indicadores adelantados), seguridad alimentaria, clima. Tableros públicos que cualquiera puede ver y que se accionan automáticamente.

**Simulacros regulares.** Cada provincia ejecuta un simulacro completo al año: corte energético prolongado, brote sanitario, evento climático extremo. La preparación se vuelve cultura, no protocolo guardado.

**Planes de continuidad por provincia.** Cómo se gobierna si se rompe la conectividad nacional, cómo funcionan los servicios mínimos, cómo se coordinan las decisiones críticas. Probados, no escritos.

**Coordinación intergubernamental aceitada.** Mesa permanente Nación + Provincias + Municipios para la respuesta coordinada cuando importa.

PLANCRI no quiere asustar; quiere quitar el factor sorpresa al miedo.
